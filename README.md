# 🚀 System & GPU Optimizer: Extreme Edition
**Created by Trigger911** | **Version: 03-15-2026**

I got tired of Windows turning off my USB ports, introducing micro-stutters, and over-managing my hardware. This script is my personal "one-click" solution for power users, sim-racers, and handheld gamers (Legion Go, ROG Ally) who want their hardware to stay **on** and stay **fast**.

---

## **🔥 Key Features**
*   **Comprehensive Bloatware Purge:** Removes 20+ junk apps including Xbox, Spotify, Teams, and Copilot.
*   **Handheld FPS Boost:** Hard-disables **VBS (Virtualization-Based Security)** and **Core Isolation** for significant CPU overhead gains.
*   **Zero USB Disconnects:** Prevents Windows from putting USB Hubs and controllers to sleep.
*   **Optional "Just the Browser" Tweak:** Interactive choice to strip down Chrome, Edge, and Firefox to bare essentials for privacy and speed.
*   **Smart RAM Management:** Detects 16GB+ RAM to disable memory compression and optimize system services.
*   **Low Latency Network:** Resets the Winsock stack and kills "Energy Efficient Ethernet" stuttering.

---

## **🚀 How to Use**
1. **Download** `System Tweaks.bat`.
2. **Right-click** and select **Run as Administrator**.
3. Follow the **12 automated steps**.
4. **Restart your computer** to apply hardware-level changes.

---

## **🎮 NVIDIA Driver Optimization**
For NVIDIA users, I’ve included `New_TriggersNvidiaProfile.txt`.
1. Open **NVIDIA Profile Inspector**.
2. Click **Import Profiles** (Green arrow icon).
3. Select my `.txt` file and click **Apply Changes**.
